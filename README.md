#### 📖 Git Commit & Branch naming Convention
| 이모지 | 타입 | 상황 |
| --- | --- | --- |
| 🎨 | design | CSS 등 사용자 UI 디자인 변경 |
| ✨ | feat | 새로운 기능 추가, 구현  |
| 📝 | docs | 문서 파일 추가 및 수정 (readme) |
| 🚚 | docs | 파일, 경로, route를 옮기거나 이름 변경 |
| ✏️ | chore | 단순 오타 수정 |
| 🚨 | fix | 버그 수정 |
| ♻️ | refactor | 코드 리팩토링 |
| 🗑️ | remove | 삭제 (파일, 코드) |
